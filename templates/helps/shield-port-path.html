Raspberry Pi / Linux Version

USB / Serial ports usually listen at one of these address:
 /dev/ttyACM0
 /dev/ttyACA0
 /dev/ttyUSB0

To find out correct address you can connect to Raspberry through SSH protocol using tools such as Putty. Run following command and take notes of available paths.

ls /dev/tty*

Connect your device to Raspberry Pi and type above command again. Additional path is belong to your device.


Windows Version

Enter RAMPS COM port address.
Steps to find correct COM port on windows.
  1. Open the "Run" dialog box by pressing and holding the Windows key, then press the R key ("Run").
  2. Type devmgmt.msc and press enter.
  3. Check out Ports section of the device manager.
