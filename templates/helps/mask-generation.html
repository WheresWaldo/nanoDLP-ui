You need UV sensor or equivalent to use this feature.

How it works

  1. Empty the vat of resin and clean the vat bottom surfaces inside and out
  2. Specify number of measurement points in both x and y axis, and the size of measurement squares
  3. Tabular cells are displayed on the user interface. You can change the cell values from 0=black to 255=white.
  4. When measuring, hold the UV sensor in the middle of the measurement square projected on the inside the vat
  5. Use the UV sensor to determine a set point value which is usually the lowest value of all the measurement squares (probably the corner values)
  6. Decrease the cell value for each measurement square until the UV sensor reads the set point value (do not press enter unless you want to display the mask)
  7. Press preview and double check the surface area to see if UV intensity is the same across the entire vat bottom.
  8. Save mask (It takes up to a minute to generate a complete mask).
  9. Goto Setup..Projector Mask to see the saved mask on the user interface (press refresh if necessary)
